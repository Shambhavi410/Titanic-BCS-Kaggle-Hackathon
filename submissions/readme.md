## SUBMISSION GUIDLINES
1. Create a fork of this repo. This creates a copy of this repository in your own GitHub account. You will make all your changes there first.
   
2. Create Your Folder
Navigate to the submissions/ folder in this repository.
Create a new folder inside it named exactly in this format: YourName_RollNumber.

3. Add Your Files Inside your personal folder, upload the following two files:
- The Notebook (.ipynb): Your code file containing the solution.
- A README.md file: A text file containing your submission details.

4. Commit changes and open a pull request
