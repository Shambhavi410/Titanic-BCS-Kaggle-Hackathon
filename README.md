# Titanic-BCS Kaggle Hackathon
Submission repository for the Titanic Hackathon. 

## How to Submit
To avoid file conflicts, please follow this structure strictly:

1.  **Create a Folder:** inside the `submissions/` directory named `YourName_RollNumber` (e.g., `submissions/Shambhavi_240958`).
2.  **Upload Files:** Put your Jupyter Notebook (`.ipynb`) inside that folder.
3.  **Add a README:** Create a `README.md` file inside your folder with your stats and approach.
